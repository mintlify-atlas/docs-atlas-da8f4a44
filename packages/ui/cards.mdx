## Card (Base Component)

The foundational card component that serves as a container for other content.

### Basic Usage

```tsx
import { 
  Card, 
  CardHeader, 
  CardTitle, 
  CardDescription, 
  CardContent,
  CardFooter 
} from "@repo/ui";

function Example() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Card Title</CardTitle>
        <CardDescription>Card description goes here</CardDescription>
      </CardHeader>
      <CardContent>
        <p>Card content</p>
      </CardContent>
      <CardFooter>
        <button>Action</button>
      </CardFooter>
    </Card>
  );
}
```

## LongCard

A horizontal card designed for list views with hover effects.

### Props

<ParamField path="variant" type="string" default="default">
  Visual variant of the card.
  
  Options: `default`, `hover`, `selected`
</ParamField>

<ParamField path="hoverable" type="boolean" default="true">
  Enable hover effect that changes background color.
</ParamField>

### Usage

```tsx
import { LongCard } from "@repo/ui";

function UserList() {
  return (
    <div className="space-y-2">
      <LongCard hoverable>
        <div className="flex-1">
          <h3 className="font-semibold">John Doe</h3>
          <p className="text-sm text-muted-foreground">john@example.com</p>
        </div>
        <button>View</button>
      </LongCard>
      
      <LongCard variant="selected">
        <div className="flex-1">
          <h3 className="font-semibold">Jane Smith</h3>
          <p className="text-sm text-muted-foreground">jane@example.com</p>
        </div>
        <button>View</button>
      </LongCard>
    </div>
  );
}
```

## MetricCard

Display key metrics and statistics with optional icons and actions.

### Props

<ParamField path="icon" type="ReactNode">
  Icon element to display at the top of the card.
</ParamField>

<ParamField path="title" type="string" required>
  The metric title/label.
</ParamField>

<ParamField path="value" type="string | number" required>
  The metric value to display prominently.
</ParamField>

<ParamField path="description" type="string">
  Additional description or context below the value.
</ParamField>

<ParamField path="action" type="ReactNode">
  Action element (usually a button or link) displayed at the bottom.
</ParamField>

### Usage

```tsx
import { MetricCard } from "@repo/ui";
import { Users } from "lucide-react";

function Dashboard() {
  return (
    <div className="grid grid-cols-3 gap-4">
      <MetricCard
        icon={<Users className="size-6" />}
        title="Total Users"
        value="2,543"
        description="+12% from last month"
      />
      
      <MetricCard
        title="Revenue"
        value="$45,231"
        description="+8% from last month"
        action={<a href="/revenue" className="text-sm text-primary">View details</a>}
      />
    </div>
  );
}
```

## StatCard

Compact card for displaying a single statistic with an icon.

### Props

<ParamField path="label" type="string" required>
  The stat label.
</ParamField>

<ParamField path="value" type="string | number" required>
  The stat value.
</ParamField>

<ParamField path="icon" type="ReactNode">
  Icon element to display.
</ParamField>

<ParamField path="iconColor" type="string" default="text-primary">
  Tailwind color class for the icon.
</ParamField>

<ParamField path="iconBg" type="string" default="bg-primary/10">
  Tailwind background class for the icon container.
</ParamField>

<ParamField path="suffix" type="string">
  Text to append after the value (e.g., "%", "k").
</ParamField>

### Usage

```tsx
import { StatCard } from "@repo/ui";
import { DollarSign, TrendingUp, Users } from "lucide-react";

function Stats() {
  return (
    <div className="grid grid-cols-3 gap-4">
      <StatCard
        label="Total Revenue"
        value="$12,450"
        icon={<DollarSign className="size-5" />}
        iconColor="text-green-600"
        iconBg="bg-green-100"
      />
      
      <StatCard
        label="Growth Rate"
        value="23"
        suffix="%"
        icon={<TrendingUp className="size-5" />}
        iconColor="text-blue-600"
        iconBg="bg-blue-100"
      />
      
      <StatCard
        label="Active Users"
        value="1.2"
        suffix="k"
        icon={<Users className="size-5" />}
      />
    </div>
  );
}
```

## QuickActionCard

Interactive card for triggering actions, rendered as a button.

### Props

<ParamField path="title" type="string" required>
  The action title.
</ParamField>

<ParamField path="description" type="string">
  Optional description of the action.
</ParamField>

<ParamField path="icon" type="ReactNode">
  Icon element to display.
</ParamField>

<ParamField path="iconColor" type="string" default="text-primary">
  Tailwind color class for the icon.
</ParamField>

<ParamField path="iconBg" type="string" default="bg-primary/10">
  Tailwind background class for the icon container.
</ParamField>

<ParamField path="onClick" type="function">
  Click handler function.
</ParamField>

### Usage

```tsx
import { QuickActionCard } from "@repo/ui";
import { Plus, Upload, Download } from "lucide-react";

function QuickActions() {
  return (
    <div className="grid grid-cols-3 gap-4">
      <QuickActionCard
        title="Create New"
        description="Start a new project"
        icon={<Plus className="size-6" />}
        onClick={() => console.log('Create clicked')}
      />
      
      <QuickActionCard
        title="Upload Files"
        description="Import your data"
        icon={<Upload className="size-6" />}
        iconColor="text-blue-600"
        iconBg="bg-blue-100"
        onClick={() => console.log('Upload clicked')}
      />
      
      <QuickActionCard
        title="Export Data"
        icon={<Download className="size-6" />}
        iconColor="text-green-600"
        iconBg="bg-green-100"
        onClick={() => console.log('Export clicked')}
      />
    </div>
  );
}
```

The QuickActionCard has:
- Hover effects (shadow increase, icon scale)
- Proper button semantics
- Keyboard accessibility
- Transition animations

## Styling Details

All cards share:
- **Rounded corners** - `rounded-3xl` (24px border radius)
- **Border** - Subtle border using theme colors
- **Shadow** - `shadow-sm` for depth
- **Background** - Card background color from theme
- **Padding** - Generous padding (p-6) for content breathing room
